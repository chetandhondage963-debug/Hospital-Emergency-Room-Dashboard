# Hospital Emergency Room Dashboard

## Project Overview
This project presents an interactive Hospital Emergency Room Dashboard designed to analyze patient flow, wait times, admission status, and departmental referrals. The dashboard helps hospital management monitor performance and identify improvement areas.

## Tools Used
- Microsoft Excel

## Key Insights
- Total number of patients and average wait time
- Patient satisfaction score analysis
- Admission vs non-admission status
- Patient attendance (On-time vs Delay)
- Gender-wise distribution
- Age group analysis
- Department referral trends

## Learnings
- Data cleaning and transformation
- Creating interactive slicers and KPIs
- Designing user-friendly dashboards
- Healthcare data analysis fundamentals
